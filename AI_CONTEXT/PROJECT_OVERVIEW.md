# Finance SMS POC - Project Overview

## Goal
Build an Android-only finance SMS proof-of-concept app to verify whether SMS-based finance tracking is feasible.

## Scope
- SMS permission request
- Read inbox SMS
- Display SMS list
- Filter bank SMS only

## Stack
- Expo SDK 55
- Expo Router
- React Native 0.83
- TypeScript strict mode
- Android only
- Expo Development Build
- arm64-v8a only optimized APK
- React Compiler enabled

## Constraints
- Android only.
- Minimize dependencies.
- Avoid deprecated APIs.
- Expo Development Build compatible.
- Maintain AI continuity.
